# Prompt Testing Examples

## Example 1 – Color Palette Generation
**Role**: Senior UI/UX Designer  
**Task**: Create a comprehensive color palette for a sustainable ride-sharing brand.  
**Prompt**: As a [Role], your task is to [Task] for a mobile app named 'EcoRide'. Focus on eco-friendly and high-tech vibes.  
**Input**: Brand values: Sustainability, Innovation, Trust.  
**AI Output**: 
- Primary: #10b981 (Emerald Green)
- Secondary: #3b82f6 (Electric Blue)
- Neutral: #1f2937 (Dark Slate)
**Feedback**: Output is visually consistent with brand values.

## Example 2 – Typography Scale
**Role**: Design Systems Engineer  
**Task**: Generate a responsive typography system using a specific scale.  
**Prompt**: Acting as a [Role], [Task] for a mobile-first dashboard.  
**Input**: Base font size: 16px, Scale: 1.25.  
**AI Output**:
- H1: 3.052rem (48.83px)
- Body: 1rem (16px)
**Feedback**: Accurate scaling, but H1 might need a mobile override.

## Example 3 – User Journey Mapping
**Role**: Product UX Researcher  
**Task**: Map the user journey for booking an electric vehicle.  
**Prompt**: You are a [Role]. Please [Task] to ensure a frictionless experience.  
**Input**: Goal: Book an EV in under 4 taps.  
**AI Output**:
1. Map View -> 2. Select Destination -> 3. Choose EV -> 4. Confirm.
**Feedback**: Flow is efficient and meets the tap-count constraint.

## Example 4 – Accessibility Audit
**Role**: Accessibility Specialist (WCAG Expert)  
**Task**: Conduct a color contrast audit for UI elements.  
**Prompt**: As an [Role], [Task] for the provided hex codes.  
**Input**: Text: #059669, Background: #f3f4f6.  
**AI Output**:
- Ratio: 4.58:1 (WCAG AA Pass).
**Feedback**: Precise analysis, helpful for compliance.

## Example 5 – Component States
**Role**: Senior Frontend Developer  
**Task**: Define interactive CSS states for a primary button.  
**Prompt**: Acting as a [Role], [Task] for a dark-themed UI component.  
**Input**: Component: 'Confirm' Button, Base: #2563eb.  
**AI Output**:
- Hover: #3b82f6, Active: #1d4ed8, Disabled: #9ca3af.
**Feedback**: Comprehensive coverage of all necessary interaction states.
