# EcoRide Monolithic Architecture Description

The EcoRide platform follows a monolithic architecture where all business logic resides within a single codebase.

### Component Descriptions:

* **Frontend Mobile/Web App**: The client-side interface for riders and drivers.
* **EcoRide Monolithic Backend**: The central server processing all business logic.
* **Auth & User Management**: Handles registration and SSO.
* **Eco-Matching Algorithm**: Pairs riders with eco-friendly drivers.
* **Real-time GPS Tracking**: Manages live location updates.
* **Sustainability Analytics**: Calculates carbon footprint savings.
* **Payment & Green Incentives**: Processes payments and rewards.
* **Notification System**: Sends trip-related alerts.
* **Central Database**: Centralized storage for all system data.
