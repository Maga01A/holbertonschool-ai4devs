# Workflow Baseline – IDE Integration

## Current IDE Setup
- **Primary IDE:** Visual Studio Code (VS Code)
- **Key Extensions:** GitHub Copilot, Prettier, ESLint, Python (Pylance), GitLens.
- **Tools:** Terminal (Zsh), Postman for API testing, Docker Desktop.

## Pain Points (Zəif Nöqtələr)
1. **Manual Boilerplate Generation:** Hər yeni API və ya verilənlər bazası modeli üçün oxşar strukturların (CRUD) təkrar-təkrar yazılması.
2. **Context Switching for Documentation:** Kod yazarkən dokumentasiya (README və ya JSDoc) hazırlamaq üçün fokusun dağılması.
3. **Complex Logic Debugging:** Xüsusilə TradingView scriptləri və ya mürəkkəb backend məntiqlərində xətanın kökünü tapmaq üçün çoxlu print/log istifadəsi və vaxt itkisi.

## Productivity Metrics (Məhsuldarlıq Göstəriciləri)
- **Avg. Task Completion:** 120 dəqiqə (kiçik və orta ölçülü xüsusiyyətlər üçün).
- **Bug Fix Turnaround:** 3 saat (problemin aşkarlanmasından həllinə qədər).
- **Weekly Commits:** ~20-30 commit (aktiv layihə dövründə).
