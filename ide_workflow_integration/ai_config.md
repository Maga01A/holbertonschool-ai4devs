# AI Configuration – IDE Workflow

## Setup Details
- **AI Tool:** GitHub Copilot / Copilot Chat
- **Configuration Method:** `.copilot-settings.yaml` (Project-specific rules)
- **Integration:** VS Code Workspace Settings

## Specialized Workflows
1. **Automated Code Review:** AI is configured to check for security vulnerabilities and performance bottlenecks before every commit.
2. **Documentation Generator:** A specialized prompt workflow for generating consistent JSDoc and Python Docstrings.

## Language Rules
- **Python:** Strict type hints and adherence to the Black formatter.
- **JavaScript:** Airbnb style guide with mandatory semicolons.
