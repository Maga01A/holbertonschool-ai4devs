# UI Concept - EcoRide## Purpose
Help users book eco-friendly rides and track their carbon footprint reduction.## Users
- Eco-Riders: Individuals looking for electric vehicle transportation.
- EV Drivers: Owners of electric cars providing sustainable rides.## Core Screens
- Rider Dashboard with interactive map
- Sustainability Profile with CO2 statistics
- Trip History with environmental impact details## Actions
- Book an Eco-Ride
- View total CO2 saved
- Manage eco-reward points
