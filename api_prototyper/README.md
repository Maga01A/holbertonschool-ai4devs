# RentFlow API Prototype
A minimal working API for managing rental equipment, built with Express.js.

## Setup Instructions
1. Navigate to the project folder: cd api_prototyper
2. Install dependencies: 
pm install express
3. Start the server: 
ode src/index.js

## API Endpoints (CRUD)
- **GET** /items: All rental items
- **POST** /items: Add a new item
- **GET** /items/:id: Get a specific item
- **PUT** /items/:id: Update an item
- **DELETE** /items/:id: Delete an item
