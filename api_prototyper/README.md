# RentFlow API Prototype
A minimal API for managing rental equipment.

## Setup
1. Navigate to directory: cd api_prototyper
2. Install dependencies: 
pm install express
3. Run server: 
ode src/index.js

## Endpoints (CRUD)
- GET /items: List all equipment
- POST /items: Add new equipment
- GET /items/:id: Get item details
- PUT /items/:id: Update item info
- DELETE /items/:id: Remove equipment
