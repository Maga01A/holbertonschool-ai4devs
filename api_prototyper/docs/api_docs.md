# EcoRide & RentFlow Unified API
## Endpoints Covered:
### Users Resource:
- GET /users (List)
- POST /users (Create)
- GET /users/:id (Read)
- PUT /users/:id (Update)
- DELETE /users/:id (Delete)

### Items Resource:
- GET /items (List)
- POST /items (Create)
- GET /items/:id (Read)
- PUT /items/:id (Update)
- DELETE /items/:id (Delete)
